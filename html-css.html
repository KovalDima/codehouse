Программа курса frontend

Верстка Html, Css, Git

  1. Этапы создания сайта, установка и настройка редактора кода, знакомство с Figma;
  2. Html, основные теги, семантика;
  3. CSS, блочная модель, работа с Developer Tool, позиционирование и выравнивание;
  4. Селекторы CSS, их специфичность, вложенность;
  5. Создание первого сайта на Html и CSS;
  6. Технология Flexbox, работа с шрифтами;
  7. Работа с формами;
  8. CSS Grid;
  9. Библиотека Bootstrap;
  10. CSS анимации;
  11. Препроцессоры SCSS, SASS;
  12. Псевдоклассы и псевдоэлементы в CSS, работа с иконками;
  13. Адаптация проектов под мобильные устройства;
  14. Публикация сайта в интернете и его оптимизация;
  15. Система контроля версий Git;
  16. Планировщик задач Gulp, методология БЭМ;
  17. Знакомство с JavaScript, интеграция JS в проект;
